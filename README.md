## ðŸ“¦ Deployment to GitHub Pages

1.  **Configure Astro**:
    Open `astro.config.mjs` and add your site URL if known (optional but recommended for sitemap).

2.  **Build**:
    ```bash
    npm run build
    ```

3.  **Deploy**:
    This project is ready for GitHub Pages.
    - Push code to GitHub.
    - Go to Settings > Pages.
    - Select `GitHub Actions` as the source, or use the standard `npm run build` workflow.
    
    *Manual Deployment*:
    Push the contents of the `dist/` folder to a `gh-pages` branch.
